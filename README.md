# House Price Prediction using Linear Regression

## Objective

Build a Linear Regression model to predict house prices using:

* GrLivArea (Living Area)
* BedroomAbvGr (Number of Bedrooms)
* FullBath (Number of Bathrooms)

## Dataset

House Prices - Advanced Regression Techniques (Kaggle)

Target Variable:

* SalePrice

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Project Workflow

1. Load Dataset
2. Data Exploration
3. Data Visualization
4. Feature Selection
5. Train-Test Split
6. Linear Regression Model Training
7. Prediction
8. Model Evaluation

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

## Visualizations

* House Price Distribution
* Living Area vs Sale Price
* Correlation Heatmap
* Actual vs Predicted Prices
* Residual Plot

## Conclusion

The Linear Regression model successfully predicts house prices using housing features such as living area, bedrooms, and bathrooms.
